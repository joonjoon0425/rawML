\section{Notations}
\begin{itemize}
    \item \textbf{x}: vector
    \item \textbf{X}: matrix
\end{itemize}
Suppose that the \textbf{X} consists of vector components $\textbf{x}_1, \textbf{x}_2, \dots, \textbf{x}_n$. Then,
    \[
        \mathbf{X} =
        \begin{bmatrix}
            \mathbf{x}_1^\top \\ \vdots \\ \mathbf{x}_N^\top
        \end{bmatrix}
    \]
